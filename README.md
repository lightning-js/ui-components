# Lightning UI

This library contains shared Lightning components, updated for @lightningjs version 3

## Package List

in progress:

- @lightningjs/solid-ui

coming soon:

- @lightningjs/blits-ui

## Development

Run the local dev environment:

```sh
> pnpm install
> pnpm start
```

If you don't have pnpm
```sh
> npm install -g pnpm
```
